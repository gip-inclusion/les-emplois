{% component_info c_info__summary=c_info__summary c_info__detail=c_info__detail collapse_id="collapseDoubleAuth" extra_classes="mb-3 mb-md-4" %} {% fragment as c_info__summary %} Qu’est-ce que la double authentification ? {% endfragment %} {% fragment as c_info__detail %}

Votre mot de passe seul ne suffit plus à protéger votre compte. S’il est découvert ou volé, n’importe qui pourrait accéder aux données des personnes que vous accompagnez.

La double authentification ajoute une deuxième vérification au moment de la connexion. En plus de votre mot de passe, on vous demande un code à 6 chiffres généré par une application sur votre téléphone ou votre gestionnaire de mots de passe sur votre ordinateur. Ce code change toutes les 30 secondes et n’est valable qu’une seule fois.

Concrètement, à chaque connexion :

  1. Vous saisissez votre identifiant et votre mot de passe.
  2. Vous ouvrez votre application ou votre gestionnaire de mots de passe et recopiez le code affiché.
  3. Vous accédez à votre espace.

Même si quelqu’un obtient votre mot de passe, il ne peut pas se connecter sans avoir accès à votre appareil.
