Installez une application d’authentification sur votre téléphone

{# FIXME (dbaty): add picture or animation here when ready #}
  1. Ouvrez l’App Store (iPhone) ou le Play Store (Android) sur votre téléphone.
  2. Dans la barre de recherche, tapez le nom de l’application de votre choix. Plusieurs applications sont compatibles, par exemple Google Authenticator (Google LLC), Authy (Twilio) ou Microsoft Authenticator (Microsoft Corporation).
  3. Sélectionnez l’application dans les résultats de recherche.
  4. Appuyez sur le bouton « Obtenir » (iPhone) ou « Installer » (Android).
  5. Attendez la fin du téléchargement, puis appuyez sur « Ouvrir ».
{% component_alert title=title content=content variant="info" %} {% fragment as title %} Cette application est isolée du reste de votre téléphone. {% endfragment %} {% fragment as content %}
  • Elle ne lit ni vos messages, ni vos photos, ni vos contacts.
  • Elle génère uniquement des codes chiffés pour sécuriser votre connexion.
{% endfragment %} {% endcomponent_alert %}

Activez votre gestionnaire de mots de passe

{# FIXME (dbaty): add picture or animation here when ready #}
  1. Installez un gestionnaire de mots de passe compatible sur votre ordinateur : Bitwarden (gratuit), 1Password ou Dashlane.
  2. Créez un compte sur le site du gestionnaire choisi.
  3. Une fois connecté, créez une nouvelle entrée pour Les emplois de l’inclusion en y renseignant votre identifiant et mot de passe.
  4. Dans cette fiche, cherchez l’option « Authentification à deux facteurs », « Code TOTP » ou « One-Time Password » selon le gestionnaire.
  5. Copiez la clé de configuration qui vous sera proposée à l’étape suivante.
  6. Votre gestionnaire génèrera désormais les codes de validation à usage unique directement depuis votre ordinateur.

  • En cas de doute, contactez votre service informatique ou notre support.
