Nous développons progressivement nos connexions avec les différents services de l’État qui fournissent les justificatifs correspondant aux critères administratifs. Après interrogation du système d’information de l’État, plusieurs types de badges peuvent être affichés :

{% include "eligibility/includes/badge_certified.html" only %}
Aucune action n’est nécessaire.
{% include "eligibility/includes/badge_not_certified.html" only %}
Ce critère n’est pas reconnu par les services de l’État pour cette personne à ce jour.
{% include "eligibility/includes/badge_certification_in_progress.html" only %}
La vérification de ce critère est en cours.
{% include "eligibility/includes/badge_certification_error.html" only %}
Ce critère n’a pas pu être certifié (profil utilisateur incomplet, utilisateur introuvable, erreur serveur).

Si aucun badge n’apparaît, cela signifie que la vérification de ce critère n’est pas encore disponible.

Pour rappel :

  • les critères déclarés par les prescripteurs habilités ne nécessitent aucun justificatif.
  • Lorsqu’un critère n’est pas certifié dans le cadre d’une auto-prescription, l’employeur doit donc récupérer et conserver une copie du justificatif administratif.

Retrouvez le détail des conditions dans notre documentation.
